# Pet4U Pet Shelter Website Prototype
## Overview

A high-fidelity interactive and clickable prototype designed to visualize what a comprehensive and intuitive pet shelter website would look like. The project aims to streamline the pet adoption process by making it more accessible and easier to use for both administrators and adopters.
## Features

- Image Carousel / Slideshow
- Navigation Bar
- Interactive form fields that allow input (somewhat)
- 2FA Authentication 
- Login and Register
- Filter pets by type and scroll to next page
## Tools Used
- Figma
- paint.net
